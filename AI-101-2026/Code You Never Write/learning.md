# Prompts

1. ```text
    Open the file.
    For each row, read the category and the amount.
    Add the amount to the category's running total.
    When done, Print every total
   ```

2. ```text

   Write and Run code to
   [total these number then tell me the average and how many are above 40: 40: 12, 47, 8 93, 21, 56],
   Show me the code you run.
   ```

3. To avoid Skimming

```text
Write and run code to [your task]. Show me the code you run.
```

# Concepts

1. Answer from Knowledge vs Answer from Running Code.

- These are answered from Knowledge

```text
Draft, Advise, Summarize, Explain, Brainstorm, Judge an idea
```

- These are answered from Running Code

```text
Total, Merge, Scan, Rename, Clean, Reconcile, Chart, Automate
```

2. VPRF(Volume, Precision, Repetition, Files)

3. Differentiate b/w Code Problem and Answer Problem.
